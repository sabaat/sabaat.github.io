I am a **Ph.D. candidate in Computer Science at Virginia Tech**, working with [Dr. Muhammad Ali Gulzar](https://mgulzar.github.io/) in the DSSL lab.

My recent research and systems work includes:

- **Assessing the Impact of Code Changes on the Fault Localizability of Large Language Models** — Evaluating the debugging capability of LLMs under realistic scenarios of code evolution over 750,013 fault-localization tasks in Java and Python. This study shows that LLMs fail to localize the same fault after non-functional changes in code in 78% of cases.

- **DeSQL: Interactive Debugging of SQL in Data-Intensive Scalable Computing** — Automated, fine-grained query debugging for Apache Spark SQL with minimal overhead, using query decomposition, taint analysis, and data reuse. DeSQL debugging time is **7.4× less** than naïve debugging time.

- **Evaluating LLM-Based Test Generation Under Code Changes** — Large-scale study of LLM test generation robustness under code evolution across **22,374 evolved program variants**.

Beyond these projects, I have **four years of hands-on Apache Spark** experience—from **core runtime** internals to **Spark SQL extensions** for **large-scale analytics** workloads. I also have over 3 years of experience designing large-scale evaluation frameworks for LLM-based code reasoning, focusing on fault localization, code understanding, and test generation. My publications appear in **ESEC/FSE**, **IEEE ICST**, and **IEEE Big Data**, with additional **arXiv** preprints.

I received my **M.S. in Computer Science from Virginia Tech** (2024) and my **B.S. in Computer Science from FAST NUCES, Lahore** (Graduated Cum Laude, 2020).

Since **August 2022**, I have served as a **Graduate Teaching Assistant** at Virginia Tech, supporting **CS 5704 (Software Engineering)**, **CS 5614 (Big Data Engineering and Database Management Systems)**, and **CS 2506 (Computer Organization II)** across multiple terms. My work includes **recitations**, **office hours and student mentoring**, **assessment design** (rubrics, quizzes, and exams), and **LMS coordination**. I am applying to **lecturer** roles in computer science where I can combine inclusive undergraduate instruction with strong technical depth in software systems and data-intensive computing. A term-by-term course list is on my **[Teaching](/teaching/)** page.
