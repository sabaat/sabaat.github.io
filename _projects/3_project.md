This project studies the reliability of LLM-generated tests when source code evolves over time, emphasizing practical software engineering conditions.

- Built an evaluation pipeline spanning **22,374** evolved program variants.
- Assessed how model-generated tests adapt to structural and non-functional code updates.
- Produced actionable findings for test generation workflows and classroom practice in software testing.
