This project evaluates how non-functional code changes affect the fault localization ability of large language models in realistic software maintenance settings.

- Evaluated **750,013** fault-localization tasks across Java and Python.
- Measured behavior across evolving code snapshots to test robustness and consistency.
- Found that models fail to localize the same fault after non-functional changes in **78%** of cases.
