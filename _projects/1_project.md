DeSQL is an interactive debugging system for Apache Spark SQL that improves developer productivity when diagnosing complex query failures in large-scale data workflows.

- Uses query decomposition and taint analysis to isolate error sources in SQL pipelines.
- Reuses intermediate data to reduce repeated execution overhead during debugging.
- Demonstrates **7.4x faster** debugging compared to naive workflows.
